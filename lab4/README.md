## Project overview
This project loads a weather CSV, cleans it, computes statistics, creates visualisations, groups data by season/month, and exports results and plots.

## Files to include in the repository
- `weather_analysis.py` — main script
- `weather.csv` — original dataset (rename your CSV)
- `outputs/` — generated outputs (PNG files, CSVs, report)
- `summary_report.md` — interpretive report
- `README.md` — this file
